# Shoot 'Em Up
A 2D side-scrolling shoot ’em up (shmup) built in Unity, featuring dynamic enemies, multiple ship types, and scoring mechanics.
Itch.io: https://etaranen.itch.io/shmup-game

## How to Play
- Avoid enemy attacks and survive as long as possible
- The game ends when your HP reaches 0
- Earn points by destroying enemies, surviving longer, and collecting items


## Features
- Side-scrolling movement with enemies and collectibles
- Free player movement within screen boundaries
- Projectile system with straight-line shooting
- Enemies shoot at the player and actively dodge incoming player projectiles
